# 🛒 MIZA Équipement - E-commerce Platform

### 📝 Description
Une plateforme de commerce électronique moderne développée avec **Laravel**, conçue spécifiquement pour le magasin **MIZA Équipement** à Safi. Le site permet de gérer les ventes d'électroménager et d'équipements de maison avec un tableau de bord complet.

### 🚀 Fonctionnalités
* **Interface Client :** Consultation des produits, catégories, et panier d'achat.
* **Dashboard Admin :** Gestion totale des produits (CRUD), des commandes et des stocks.
* **Design :** Responsive et moderne (Blue & White & Orange).
* **Architecture :** MVC avec Laravel.

### 🛠️ Tech Stack
* **Backend :** PHP 8, Laravel Framework.
* **Database :** MySQL.
* **Frontend :** Blade, CSS, JavaScript.
* **DevOps :** Docker (en cours), Git.
